Latitude Vs. X

The purpose fo this assignment was to use Python together with the Open Weather Map API to visualize the relationship between latitude and weather characteristics such as humidity, temperature, wind speed and cloudiness. To do this, we randomly selected over 500 unique cities based on their latitude and longitude, performed an API call on each citie's weather characteristics and then plotted the cloudiness percentage, humidity percentage, wind speed (in mph) and the temperature (F) versus the latitude of each city. In the end, using Matplotlib, we constructed 4 different scatter plots of each different weather characteristc along with the latitude for all 500 cities.
